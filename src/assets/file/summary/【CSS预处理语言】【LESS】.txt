【unit()】
【ceil()】向上取整
【floor()】向下取整
【round()】四舍五入
【percentage()】例如 percentage(0.5) 编译结果: 50%
【abs()】绝对值
 
body {
	background-color: @bgColor;
	width: unit(@width, px);
	height: unit(@height, px);
}

|第7课：
+--------------------------------------------------------------------------------------------------------------------------
【loop】

.widthFun(100);

.widthFun(@n, @i: 10) when (@i <= @n) {
	.width-@{i} {
		width: (@i * 100% / @n);
	}
	
	.widthFun(@n, (@i+10) )
}

结果
.width-10{
	width: 10%;
}

.width-20 {
	width: 20%;
}

...

.width-100 {
	width: 100%;
}
