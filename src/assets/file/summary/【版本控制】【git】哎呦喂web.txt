-------------------------------------------------------------------------------------
【git --version 】查看git版本
git --version 

-------------------------------------------------------------------------------------
【git --help】 查看命令

-------------------------------------------------------------------------------------
【git init】初始化

-------------------------------------------------------------------------------------
【git add】准备提交--暂存区
git add  .		----同级
git add -u 		----update
git add -A 		----all

-------------------------------------------------------------------------------------
【git commit】提交
git commit -m  ‘这里设置记录<注释>’		----message

-------------------------------------------------------------------------------------
【git status】查看git状态

-------------------------------------------------------------------------------------
【git reflog】查看所有版本
git reflog		----reflow： vi. 流回；再涨；重排版

-------------------------------------------------------------------------------------
【git reset】回滚版本
git reset --hard [版本号]

git reset --hard HEAD^ 		回滚到上一个版本
git reset --hard HEAD^^ 	回滚到上两个个版本

-------------------------------------------------------------------------------------
【git checkout】将暂存区的内容撤销掉
git checkout a.txt

-------------------------------------------------------------------------------------
【git rm】移除暂存区的内容, 和reset同功能
git rm --cache a.txt

-------------------------------------------------------------------------------------
git remote add origin https://github.com/Cohen-Wang/webinn.git

-------------------------------------------------------------------------------------
【git push】上传，提交
第一次，提交到master分支上：git push -u origin master
	origin 		表示远程中心

第二次就可以直接提交了, 但考虑到多人在线同时开发，所有先pull一下
	git pull
	git push

-------------------------------------------------------------------------------------
【git branch】查看项目分支
git branch

-------------------------------------------------------------------------------------
【git checkout】创建分支
git checkout -b test		创建，或切换到 一个test分支
git checkout master			切换回master主分支

-------------------------------------------------------------------------------------
【git merge】合并
git checkout master 	先切换回master主分支上
git merge test			再合并test分支上的内容

注意： 当有冲突代码时，要选择哪一边的代码        <<<<<<<<<<<<<<< --------冲突--------- >>>>>>>>>>>>>>>

-------------------------------------------------------------------------------------
【git branch -D】 删除分支
git branch -D test		删除test分支

-------------------------------------------------------------------------------------
【git tag】版本重大更新时，添加的标签
git tag v1.0.0 		添加标签为v1.0.0
git tag 			查看所有版本
git show v1.0.0		查看v1.0.0的版本的具体信息和内容
git tag -d v1.0.0	删除v1.0.0的版本
git push origin v1.0.0	提交v1.0.0


-------------------------------------------------------------------------------------
【git clone】 从远程下载项目
目标文件夹下面： git clone https://github.com/Cohen-Wang/webinn.git
