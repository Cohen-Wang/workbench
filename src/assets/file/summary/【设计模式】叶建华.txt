|1.基本类型和复杂类型
+--------------------------------------------------------------------------------------------------------------------------
基本类型：	
	栈；
	速度快，存储小，使用时间短；
	值赋值；
	
复杂类型：	
	堆栈，如：obj => 0x0000001 指向一个堆 {name: xxx, age: xxx}; 
	速度相对慢，存储打，使用时间长；
	引用赋值。或者说也是值赋值，只不过赋值的是地址，结果还是引用赋值；

|4.异常处理
+--------------------------------------------------------------------------------------------------------------------------

// 获取异常
try {
	// 可能出错的代码
} catch (e) {
	// 出错就执行的代码
} finally {
	// 不管是否出错，都会执行的代码
	// 一般前端开发中，不会使用。多用于后端开发nodejs,主要用于最后【释放资源】
}

// 抛出异常
function divide (a, b) {
	if (b === 0) {
		// 抛出
		throw new Error('分母不能为0');
	}
	return a / b;
}
