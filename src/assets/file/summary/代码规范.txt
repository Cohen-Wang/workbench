

+--------------------------------------------------------------------------------------------------------------------------
|风格
+--------------------------------------------------------------------------------------------------------------------------
【风格】
	空格
	分号
	换行
	命名


+--------------------------------------------------------------------------------------------------------------------------
|function
+--------------------------------------------------------------------------------------------------------------------------
【传参】
	function setName (name) {
		this.name = name;
	}
	// 错误
	setName('张三');
	
	// 正确
	【let name = '张三';】
	setName(【name】);
	
【单一原则】
	函数一次只做一件事情；


+--------------------------------------------------------------------------------------------------------------------------
|注释
+--------------------------------------------------------------------------------------------------------------------------
【注释】
	/**
	 * 过任意两点的直线表达式
	 * @param {Object} startPoint
	 * @param {Object} endPoint
	 * @param {Object} thirdPoint
	 * return {Number} len : 距离
	 */
	 
【函数名即注释】
	function showName () {
		return this.name;
	}


+--------------------------------------------------------------------------------------------------------------------------
|判断
+--------------------------------------------------------------------------------------------------------------------------
【=== 取代 ==】
	判断语句一定要写成全等于（===）。不要写成等于（==），不要给自己挖坑。


+--------------------------------------------------------------------------------------------------------------------------
|返回
+--------------------------------------------------------------------------------------------------------------------------
【return】
	if 语句，以后一定要在每个语句里面写个return true ，return false；
